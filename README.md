# 🔐 Banking Cloud Security System

A full-stack cybersecurity-focused authentication system with attack detection, OTP verification, IP blocking, and geo-location tracking.

---

## 🚀 Features

- 🔐 Secure Login & Registration
- 🚨 Brute-force attack detection
- 🚫 IP blocking system
- 🔑 OTP-based new device verification
- 🌍 Geo-location tracking of login attempts
- 📊 Admin dashboard with charts & logs
- 🐳 Dockerized full-stack app
- ☸️ Kubernetes-ready deployment

---

## 🛠 Tech Stack

### Frontend
- React.js
- Chart.js
- React-Leaflet (Maps)

### Backend
- Node.js
- Express.js
- SQLite

### DevOps
- Docker
- Docker Compose
